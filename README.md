# AI-Powered Cloud-Based Online Examination System

## Overview

A cloud-native online examination platform built using microservices architecture, Kubernetes, AWS services, and AI-powered proctoring.

## Features

- Live Online Examinations
- AI Cheating Detection
- Webcam Monitoring
- Auto Result Generation
- Real-Time Analytics
- WebSocket Communication
- Kubernetes Auto Scaling
- CloudWatch Monitoring
- AWS S3 Media Storage

## Architecture

User Service
Exam Service
Result Service
Notification Service
AI Monitoring Service
WebSocket Service

## Tech Stack

- React.js
- Node.js
- Express.js
- Docker
- Kubernetes
- MongoDB
- PostgreSQL
- AWS S3
- CloudWatch
- OpenCV
